# Compilation and Usage Instructions
To compile the shell, use the following command:
1. gcc -o mysh mysh.c
2. ./mysh
3. Inside mysh, you can run built-in commands:
    mysh> cd /usr/bin
    mysh> pwd
    Or any external program:
    mysh> ls -la

# List of Implemented Features
